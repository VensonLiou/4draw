# 4Draw

4Draw is a decentralized lottery game featuring a smart contract for transparent operations, a user-friendly UI for ticket purchases and prize claims, and integration with Pragmaâ€™s VRF contract to ensure fairness.

## Frontend

### How to Start

```bash
git clone https://github.com/VensonLiou/4draw.git

cd frontend

yarn install

yarn dev
```

## How to run production

```bash
yarn build

yarn start
```

### How to update (subtree)

```bash
git subtree pull --prefix=frontend <path-to-frontend-repo> main
```
